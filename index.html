🛠️ Developer API

Built for Builders

A clean REST API with SDKs in JS, Python, and Go. Integrate WinkLink into your app in minutes.

JavaScript
// Install: npm install winklink-sdk import WinkLink from 'winklink-sdk'; const wl = new WinkLink({ apiKey: 'wl_live_••••••••••••' }); // Shorten a URL const link = await wl.links.create({ url: 'https://your-very-long-url.com/path', alias: 'my-campaign', expiresAt: '2025-12-31' }); console.log(link.shortUrl); // → "winklink.co/my-campaign" // Fetch analytics const stats = await wl.links.analytics(link.id); console.log(stats.clicks); // → 4821
